# Vendored library

`pdf-lib.esm.min.js` — [pdf-lib](https://github.com/Hopding/pdf-lib) v1.17.1,
the ESM build, used browser-side to fill the AcroForm character-sheet template
(`assets/pdf/shadowdark-character-sheet.pdf`) with actor data.

pdf-lib is MIT licensed (© Andrew Dillon). It is self-contained (no runtime
dependencies) and is loaded via a dynamic `import()` from
`pdf-sheet-export.mjs`. Do not edit — replace wholesale to upgrade.

The template PDF and its field manifest are generated by
`dev/pdf-sheet/generate-sheet.mjs`.
